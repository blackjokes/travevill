Добро пожаловать в репозиторий проекта **TraveVill**! Это SMP Minecraft-сервер, созданный для любителей классического ванильного выживания в дружной атмосфере.

> **Создатель репозитория:** `blackjokes` — член администрации TraveVill.

---

## 📅 Навигация по сезонам проекта
В этом репозитории собрана история и визуальное оформление наших сезонов. Вы можете посмотреть интерактивные слайдеры для каждого этапа:

*   [🌱 Сезон 1](https://github.io) — Начало пути, первые постройки и зарождение комьюнити.
*   [🔥 Сезон 2](https://github.io) — Масштабные проекты, новые игроки и глобальные ивенты.
*   [🛡️ Сезон 3](https://github.io) — Текущий сезон: улучшенная ванила, новые правила и города.

---

## 🛠️ О сервере TraveVill
Наш сервер ориентирован на чистое выживание без токсичных плагинов, приватов командами и лишнего мусора в чате. Всё строится на доверии и ролевом взаимодействии.

*   **Версия игры:** Изучить актуальную в чате / дискорде
*   **Режим:** Survival (Выживание)
*   **Тип сервера:** SMP (Survival Multiplayer)
*   **Доступ:** По заявкам / Вайтлист

---

## 📂 Структура этого репозитория
Для поддержания идеального порядка файлы распределены по следующим директориям:
*   `travevill-1/` — Медиаматериалы, скриншоты и слайдер первого сезона.
*   `travevill-2/` — Материалы второго сезона проекта.
*   `travevill-3/` — Контент третьего сезона.
